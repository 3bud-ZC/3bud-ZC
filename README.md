## Selected Projects

### RepoRadar AI

A local developer tool that scans repositories and generates structured project intelligence, architecture reports, security notes, README suggestions, fix plans, GitHub issues, LinkedIn posts, portfolio case studies, and AI-agent prompts.

**Stack:** TypeScript, Node.js, CLI tooling, local analysis, AI-agent workflows  
**Focus:** developer tools, repo intelligence, no paid AI, local-first scanning

---

### Abud Shorts Engine

A local-first Docker tool for generating branded short videos using templates, TTS, automatic captions, Pexels background footage, Remotion rendering, and a React UI.

**Stack:** React, TypeScript, Vite, Node.js, Express, Docker, Remotion, FFmpeg, Whisper.cpp  
**Focus:** video automation, content generation, creator tools, local-first workflow

---

### Mohamy Phone

An Android legal-office management app built for lawyers to manage cases, clients, sessions, tasks, files, local search, templates, and updates with a strong local-first and privacy-aware architecture.

**Stack:** Kotlin, Jetpack Compose, Room SQLite, Android, local-first architecture  
**Focus:** legal-tech, offline-first data, privacy, Arabic legal workflows

---

### ABUD Platform

My personal platform and portfolio for presenting services, projects, blog content, quote requests, and digital product work.

**Stack:** Next.js, TypeScript, Prisma, PostgreSQL, Tailwind CSS  
**Focus:** portfolio, services, admin structure, business presentation

---

### ERP System

Arabic ERP-style system for managing store operations, including sales, customers, inventory, invoices, reports, and operational dashboards.

**Stack:** Next.js, TypeScript, Prisma, PostgreSQL, Tailwind CSS  
**Focus:** business systems, Arabic RTL UI, operations management

---

### VPS Monitor Bot

Telegram-based monitoring assistant for tracking VPS CPU, RAM, and disk usage and sending alerts.

**Stack:** Python, Telegram Bot API, Linux VPS, system monitoring  
**Focus:** infrastructure monitoring, automation, server visibility

---

### PromptForge

Prompt-engineering platform for drafting, storing, improving, and reusing prompt templates for LLM workflows.

**Stack:** Next.js, TypeScript, Tailwind CSS, Node.js, AI APIs  
**Focus:** AI productivity, prompt systems, reusable workflows
